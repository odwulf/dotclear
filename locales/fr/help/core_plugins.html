Mise à jour de plugins

Lorsque de nouvelles version des plugins installés sont disponible sur Dotaddict, un onglet Mise à jour de plugins est disponible listant les plugins avec leurs versions installées et mises à jour.

Il suffit alors d'utiliser le bouton d'action "Mise à jour" en fin de ligne.
Si plusieurs mises à jour sont disponibles, un bouton d'action en bas de page permet de "Mettre à jour tous les plugins" en une seule fois.

Plugins installés

En début de ligne, on retrouve le logo du plugin que vous retrouverez dans la menu (à gauche de votre écran).

La présence d'un logo Dotclear signifie que le plugin est uniquement disponible avec la distribution officielle et qu'il est un de ses composants.

Plugins activés

Cette liste permet de visualiser des informations (nom, version, courte description) des plugins activés. Deux actions sont possibles : "Désactiver" (opération temporaire) ou "Supprimer" (opération irrévocable).

Plugins désactivés

Cette liste permet de visualiser les plugins ayant été désactivés par l'utilisateur. Deux actions sont possibles : "Activer" ou "Supprimer" (opération irrévocable).

Ajouter des plugins

Cette page vous permet d'installer directement un plugin disponible sur DotAddict.

Deux moyens de recherche vous sont possibles :

  • En saisissant son nom dans le champ Rechercher dans le dépot ou en parcourant l'index.
  • En déroulant la flèche, vous pouvez accéder à sa fiche DotAddict ou être diriger vers un lien de support décidé par l'auteur.

Installer ou mettre à jour manuellement

Cette page vous permet d'installer des plugins personnels ou disponible ailleurs que sur DotAddict.

Deux moyens d'installation vous sont possibles :

  • Déposer un fichier zip depuis votre ordinateur. Pour cela, utiliser le bouton "Parcourir" et dans un second temps, saisissez votre mot de passe.
  • Télécharger un fichier zip depuis une adresse URL puis saisissez votre mot de passe.
