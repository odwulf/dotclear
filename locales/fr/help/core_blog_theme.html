Mise à jour de thèmes

Lorsque de nouvelles version des thèmes installés sur votre blog sont disponibles sur Dotaddict, la page comporte un onglet supplémentaire nommé Mise à jour de thèmes listant les thèmes avec leurs versions installées et mises à jour.

Il suffit alors d'utiliser le bouton d'action "Mise à jour" en fin de ligne.
Si plusieurs mises à jour sont disponibles, un bouton d'action en bas de page permet de "Mettre à jour tous les thèmes" en une seule fois.

A noter que les modifications que vous auriez pu apporter aux fichiers du thème seront perdues lors de sa mise à jour.

Thèmes installés

La liste des thèmes installés apparaît dans cet onglet. Vous pouvez sélectionner un thème en cliquant sur "Utiliser celui-ci".

Le thème courant peut être modifié via le bouton "Modifier les fichiers du thème". Si le thème sélectionné inclut un configurateur, il est possible de personnaliser facilement ce thème via le bouton "Personnaliser le thème".

Vous pouvez également désactiver un thème (s'il pose un problème de compatibilité par exemple). Un onglet "Thèmes désactivés" apparaît alors avec la liste de ceux-ci.

Ajouter un thème

Cet onglet permet de parcourir le centre de ressources DotAddict des thèmes soit par ordre alphabétique, soit en entrant une recherche dans le champ dédié. Les thèmes déjà installés n'apparaissent pas.

Installer ou mettre à jour manuellement

Si le thème que vous voulez installer (ou mettre à jour) ne se trouve pas sur le centre de ressources DotAddict, vous pouvez l'installer soit en allant le chercher sur votre disque dur, soit en entrant son adresse URL que vous aurez préalablement récupéré sur le site qui le propose. Le thème est toujours sous la forme d'un fichier zip.
