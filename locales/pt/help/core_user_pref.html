Preferências do utilizador

Apelido, Nome
Se o campo "Alcunha" está vazio, o nome do autor exibido será o seus apelido e nome, conforme aqui.
Alcunha
Pode escolher aqui o nome de autor que será exibido nas suas entradas.
Email
Endereço para onde serão enviadas as notificações de novo comentário.
URL
Indica o website do utilizador. Se indicado, o nome de autor será apresentado como um link para este website.
Formato preferido de edição
Escolha da sintaxe para a edição.
Wiki é uma sintaxe simplificada que é convertida automaticamente em código xhtml válido ;  recomendamos a escolha da sintaxe wiki, A menos que você tenha uma perfeita compreensão do html.
Estado initial das entradas
O estado inicial das entradas pode ser determinado como:
  • pendente o estado da publicação está sendo avaliado
  • programado a entrada será publicada na hora e data definidas no campo Publicado em.
  • não publicado a entrada continua offline.
  • publicado a entrada é online.
Entry edit field height
The height of the Entry edit field. Defaults to 24.
Língua do utilizador
A língua das publicações. Se uma tradução Dotclear existe, a interface também será exibida nessa língua.
Fuso horário
Essa escolha vai definir a hora exibida nas publicações.
Activa a interface WYSIWYG
Permitir o uso do editor WYSIWYG para as entradas e descrição de campos.
WYSIWYG = What You See Is What You Get / O que você vê é o que você obtém
Alterar sua password

Para alterar sua password, digita-a duas vezes nos campos Nova password e Confirmar password. Um mínimo de 6 caracteres é requerido.

Se nenhuma nova password é fornecida, o actual é mantido.
