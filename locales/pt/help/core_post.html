Editar uma entrada

Título da entrada
Digite aqui o título da entrada. Este campo é obrigatório.
Extracto
O conteúdo deste campo será exibido nas páginas mostrando uma lista de entradas, como a página inicial ou a páginas de categorias, seguido por o link "Continuar leitura".
O conteúdo deste campo, também, será visível no início da página que mostra a entrada inteira.
Se você deixá-la em branco, o campo Conteúdo será exibido integralmente a partir da página inicial.
Conteúdo
O conteúdo da entrada. Este campo é obrigatório.
Notas
Este campo de texto é para as notas de uso pessoal. O que você entra aqui nunca será visível no blog.
Categoria
A categoria da entrada. Você pode optar por deixar a entrada fora das categorias, seleccionando a linha em branco.
Para criar uma nova categoria, por favor consulte a secção "categorias"
Estado da entrada
Permite-lhe escolher entre os diferentes estado de publicação:
  • pendente o estado da publicação está sendo avaliado (continua offline)
  • programado a entrada será publicada na hora e data definidas no campo Publicado em.
  • não publicado a entrada continua offline.
  • publicado postar a entrada online.
Publicado em
Aqui você pode mudar a data e hora de publicação.
Se o estado da entrada é programado a publicação será efectivá na referida data e hora.
Formato do texto
Escolha da sintaxe para a edição.
Wiki é uma sintaxe simplificada que é convertida automaticamente em código xhtml válido
Nos recomendamos a escolha da sintaxe wiki, a menos que você tenha uma perfeita compreensão do html.

Consulte a "Wiki syntax reference / Referência da sintaxe wiki" para mais informações.
Permitir comentários
Seleccione esta caixa para permitir aos seus visitantes comentar a entrada.
A correspondente definição global pode ser encontrada na "Configurações do blog".
Permitir trackbacks
Um trackback é uma forma de permitir que uma parte da sua entrada seja usada como comentário num outro blog. Seleccione esta caixa para permitir que outros façam um trackback da sua entrada.
A correspondente definição global pode ser encontrada na "Configurações do blog".
Entrada seleccionada
As Entradas seleccionadas serão listadas no menu do blog, sob o título "O melhor de mim".
Password da entrada
Você pode usar uma password para a sua entrada. Uma entrada protegida com password não irá ser exibida em seu blog, ela só será alcançável para quem conhece a URL e a password da entrada (clique no link ver a entrada para obter a URL).
Nome do ficheiro
você pode desproteger esta campo clicando sobre o seu bloqueio e definir uma outra URL para a sua entrada.
Se a URL escolhida já está usada por outro entrada, um número será incrementado nela.
Língua da entrada.
O código linguístico da entrada, é predefinidos pelo parâmetro da língua do utilizador, mas você pode alterá-lo para qualquer outro código linguístico válido, por exemplo "en", "pt-br" ou "fr-ca". Este código é utilizado na exibição da entrada.
Anexos
Os anexos são todos os médias associados à uma entrada através do link "Adicionar ficheiros nesta entrada".
Não se esqueça de gravar suas alterações antes de clicar neste link.

Consulte a página "Media manager / Gestor de multimédia" para mais informações.

Comentários

Lista de comentários
Aqui pode ler e alterar o estado dos comentários da entrada.
Se o vosso nível de permissão é suficiente, poderá definir os comentários como online (publicado) ou offline (não publicado), marcar-los como spam ou apagar-los.
Adicionar um comentário
Aqui, pode responder a um comentário directamente sem passar pelo seu blog. Pode aqui usar a sintaxe "texto simples" ou xHTML, sem limitação. Utilize este campo como se estivesse a modificar um comentário.

Trackback

Para criar um trackback, clique no link Pingar blogs. Você não verá este link enquanto a sua entrada não é definida como publicada.

URLs a pingar
Preencher aqui com as URLs encontradas nas entradas que você quer pingar.
Enviar extracto
Este campo contém as primeiras linhas da sua entrada, é este extracto que será enviado para o blog pingado, com um link para a entrada completa.
Você pode modificar o extracto enviado ao editar este campo.
Procura automático das URLs para ping
Se a sua entrada inclui links para umas entradas específicas e se o blog destinatário lhe permitem, esta ferramenta irá encontrar as URLs a pingar para você.
