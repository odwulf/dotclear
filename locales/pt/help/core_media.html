Gestor de multimédia

Lista de media

A página principal do gestor de multimédia exibe a lista de ficheiros disponíveis (os medias) para o blog actualmente seleccionado.

Basta clicar nos ícones ou nomes do media na pasta actual ou qualquer sub-pasta para exibi-los.

Seleccionar um ficheiro MP3 abra imediatamente o leitor de multimédia. O plugin Flash deve estar instalado em seu navegador para que o leitor funcione.

As imagens são exibidas com uma miniatura quando possível.

Adicionar ficheiros

Escolha o ficheiro
Aqui você pode escolher um ficheiro local, para enviá-lo em seu blog.
Um ficheiro maior que o tamanho máximo não será enviado.
Título
Título opcional do ficheiro que deseja enviar.
Privado
Seleccione esta opção para definir o seu ficheiro como privado, ou seja, acessível apenas pelo seu dono e não acessível a partir do blog.

Nova pasta

Este formulário permite criar uma nova pasta dentro da pasta actual. Basta escolher um nome para ele, como você poderia fazer em seu próprio computador.

Detalhes do conteúdo multimédia

Esta página apresenta um conjunto de informações sobre o media seleccionado e permite executar algumas operações nele.

Nome do ficheiro
Altere aqui o nome do ficheiro seleccionado.
File title
Definir um título ou alterar o actual.
Data do ficheiro
Alterar a data do ficheiro.
Privado
Especificar que o ficheiro ficará ocultado para todos, excepto para o seu proprietário no gestor de multimédia.
Nova pasta
Permite-lhe mover o ficheiro para uma outra pasta.
Modificar o ficheiro
Com este formulário, pode substituir o ficheiro mantendo todos os atributos originais (nome, título, etc.)
Como de costume, você tem que respeitar o tamanho máximo autorizado.

Tratamento específico para imagens

Quando você adiciona uma imagem, dependendo do seu tamanho, até quatro versões diferentes serão disponibilizadas:

  • quadrado: um quadrado de 48x48 pixeis
  • miniatura: o lado maior é reduzido para 100 pixeis.
  • pequeno: o lado maior é reduzido para 240 pixeis.
  • médio: o lado maior é reduzido para 500 pixeis.
  • original: a imagem mantém-se inalterado

Adicionar ficheiros numa entrada

Pode facilmente anexar um ficheiro em suas entradas.
Primeiro, crie uma nova entrada e guardá-a.
Em seguida, clique em "Adicionar ficheiros nesta entrada" para abrir o gestor de multimédia.
Clicar no ícone [+](Anexar este ficheiro à entrada).
O ficheiro será, então, anexado a entrada.

Os anexos serão mostrados na página das entradas, como lista ao lado do conteúdo. Os ficheiros MP3 serão disponibilizados através do Flash player, para que eles possam ser facilmente lidos pelos visitantes.

Podcast e broadcast de ficheiros musicais

Todos os ficheiros anexados serão incluídos em seus feeds RSS, permitindo o fácil podcasting de qualquer tipo de ficheiros.
