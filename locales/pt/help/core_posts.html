Acções sobre as entradas seleccionadas

Se o nível do utilizador o permite, pode executar operações sobre um conjunto de entradas seleccionadas

  • publicar: publicar as entradas.
  • não publicar: definir as entradas como offline.
  • programa: agendar a publicação das entradas.
  • marcar como pendente: apagar os outros estado de publicação
  • Mudar categoria: direcciona-lo a lista das categorias, onde você poderá escolher uma nova para as entradas.
  • Modificar autor: altere o autor das entradas, digitando o ID do novo autor.
  • apagar: suprimir as entradas seleccionadas -Aviso: esta operação é irreversível-.
