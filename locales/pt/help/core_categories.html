Categorias

Lista das categorias

Criar uma nova categoria
Escolha o nome que deseja usar para sua nova categoria, em seguida, clique em "Gravar" para validar a sua criação.
Para modificá-la, clique o seu nome na lista.
reorganizar as categorias
Para alterar o ordem das categorias, desliza as categorias usando as imagens em forma de seta, é clique em "Gravar ordem".
Se você não vir as setas, pode usar o campo numerado ao lado de cada categoria.

Modificar uma categoria

Título
Escolha o nome que deseja dar a sua categoria é clique em "Gravar" para validar.
URL
DotClear cria uma URL para a categoria, tornando-a alcançável com um caminho semelhante a esta:
http://omeusuperblog.pt/categorias/minha-categoria.
Você pode clicar no pequeno cadeado a direito por modificá-la de acordo com suas necessidades.
Descrição
O que está dentro do campo descrição será exibida ao seleccionar uma das categorias do blog.
Pode preenchê-lo com qualquer tipo de conteúdo (paragrafo, listas, etc.).
O formato utilizado para esse campo é HTML.
