Wiki Syntax Referenz

Die Wiki Syntax ist ein Weg, mit minimalen Tags Deinen Text zu bearbeiten. Du benöntigst nur die Grundvoraussetzungen wie Titel, Absatz, Zitat und Liste...

Blockelemente
  • Lass eine leere Zeile zwischen zwei gleichen Blocks
  • Absatz: Freier Text, endet mit einer leeren Zeile, wenn ein nächster Absatz folgt.
  • Titel: !!! title, !! title oder ! title, was Dir die Möglichkeit gibt, 3 Formen zu haben.
  • Horizontale Linie: ----
  • Liste: Beginne jede Zeile mit * oder # für nummerierte und unnummerierte Listen. Listenzuordnung ist bereits gemacht mit folgenden Verwendungen: 
    * item 1
** item 1.1
* item 2
*# item 2.1
...
  • Vormatierter Text: Jede Zeile muss mit einem Leerzeichen beginnen.
  • Zitat: Jede Zeile muss mit einem Semikolon beginnen (>).
Tagformate
  • Betonung: zwei Anführungszeichen ''text''
  • Starke Betonung: zwei Unterstriche __text__
  • Neue Zeile: %%%
  • Einschub: zwei Pluszeichen ++text++
  • Löschung: zwei Minuszeichen --text--
  • Link: [url], [name|url], [name|url|language] oder [name|url|language|title]
  • Bild: ((url|alternative text)), ((url|alternative text|position)) oder ((url|alternative text|position|long description)).
    Die Position kann L oder G (links), R oder D (rechts) oder C (zentriert) sein.
  • Ankerpunkt: ~anchor~
  • Acronym: ??acronym|title??
  • Inline HTML: zwei backquotes ``html code``
  • Inline quote: {{quote}}, {{quote|language}} oder {{quote|language|url}}
  • Code: @@code@@
  • Fussnote: $$footnote$$
Unformatierter Text
Wenn Du Deinen Text nicht formatieren willst, dann füge einfach nur \ ein. Zum Beispiel: \[Text in eckigen Klammern ohne URL\]
HTML Code
Auch wenn Du Dich für die Verwendung der Wiki Syntax entschieden hast, Du benöntigst mitunter ausdrucksreichere Möglichkeiten, die Dir die HTML Syntax bietet. Geh folgendermassen vor: 
///html
<p style="color:red">mein text in rot</p>
///
