분류 목록

새 분류 만들기
새 분류로 쓸 이름을 쓰시고, 저장 단추를 눌러 생성결과를 확인하세요. 분류를 수정하려면, 목록에서 해당 분류의 이름을 클릭하세요.
분류 순서 바꾸기
분류 순서를 바꾸려면, 화살표 이미지를 이용해 분류를 끌어다 놓고, 순서 저장을 클릭하세요. 화살표가 보이지 않으면, 각 분류 곁의 번호가 매겨진 칸을 이용하세요.

분류 수정

제목 수정
분류 제목을 고쳐쓴 뒤, 저장 단추를 눌러 확인하세요.
URL 변경
DotClear는 법주에 기본 url을 생성합니다. 기본 url은 다음 예와 같이 브라우저에서 직접 접근할 수 있는 경로로 지정됩니다 : http://monblog.tld/category/My-category.
url칸의 오른쪽에 있는 작은 자물쇠를 누르면 편집할 수 있는 상태가 됩니다.
분류 설명
설명칸에 있는 내용은 분류를 클릭했을 때 보여지는 페이지, 즉 분류 URL에 해당하는 페이지에 분류 설명으로 나오게 되는 내용입니다. 이 내용은 문단, 목록 등 어떤 종류의 내용으로도 채워질 수 있습니다. 이 칸의 내용은 HTML 형식으로 저장됩니다.
