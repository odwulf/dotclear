폴더나 하위폴더에 있는 미디어파일을 보시려면 그 파일의 아이콘이나 이름을 클릭하시면 됩니다.

파일 추가

파일 선택
블로그로 올려놓을 로컬 파일을 선택할 수 있습니다. 최대크기보다 큰 파일은 전송되지 않습니다.
제목
보내고자하는 파일에 붙일 제목으로 선택사항입니다.
미디어 관리화면에서 감춤
이 항목은, 파일이 블로그에 공개되지 않은 한 소유자만 볼 수 있다는 것을 표시합니다. (블로그에 여러 사용자가 등록되어 있을 경우) 파일 주인이 아닌 다른 사용자는, 전체 미디어 파일관리권한을 갖고 있지 않은 한, 이 파일을 미디어관리자에서 볼 수 없습니다.
하지만 그 파일이 공개폴더에 들어있다면, 파일 주소를 알 경우 브라우저에 주소를 직접 입력함으로써 아무나 파일에 접근할 수 있다는 것에 주의하세요.

새 폴더 만들기

여기서, 현재 폴더에 하위 폴더를 만들 수 있습니다. 이를 위해서는 단지, 새 폴더의 이름을 적고 저장하기 링크를 누르면 됩니다.

미디어 상세정보

이 페이지는 현재 선택한 파일의 전체 정보를 보여주고, 그 파일에 대한 몇 가지 작업을 하실 수 있게 해줍니다.

파일 이름
현재 파일의 이름을 변경합니다.
파일 제목
파일에 대한 제목을 주거나 변경합니다.
파일 날짜
파일의 날짜를 변경합니다.
미디어관리자에서 감춤
미디어관리자에서 해당 파일 소유자만 파일을 볼 수 있도록 지정합니다. (주의. 전체 파일 관리 권한을 갖는 관리자는 소유자가 아니라도 이 파일을 볼 수 있습니다.)
새 폴더:
파일을 다른 폴더로 이동합니다.
파일 변경
현재 파일의 속성들(이름, 제목...)을 유지한 채 실제 파일을 바꿔치기합니다. 파일 크기 제한은 위와 같습니다.

이미지파일 관련 특이 사항

이미지 파일을 추가하실 때, 파일 크기에 따라 아래와 같이 최대 네 종류의 버전을 더 만들게 됩니다.

  • 아이콘: 48x48픽셀 크기의 아이콘
  • 꼬맹이사진: 세로나 가로 중, 큰 쪽이 100 픽셀인 축소이미지 (Thumbnail)
  • 작은 크기: 큰 쪽이 240 픽셀인 축소이미지.
  • 중간 크기: 큰 쪽이 500 픽셀인 축소이미지.
  • 원래 크기: 원본, 변경 안된 이미지.

글꼭지에 파일 첨부

당신의 글꼭지에 쉽게 파일을 첨부하실 수 있습니다. 먼저, 글꼭지를 생성하고 저장하세요. 그런 다음, "이 글꼭지에 파일 추가" 링크를 눌러 미디어 관리화면으로 갑니다.

거기서 [+] 아이콘을 클릭하세요 (이 파일을 글꼭지에 첨부). 그러면 파일이 글꼭지에 첨부됩니다.

첨부된 파일은 글꼭지페이지가 보일 때, 글꼭지내용에 딸린 목록으로 나타나게 됩니다. MP3 파일들은 플래시 플레이어를 통해 사용할 수 있게 되므로, 방문자가 쉽게 음악을 들을 수 있습니다.

음악파일의 포드캐스트 및 브로드캐스팅

글꼭지에 첨부된 모든 파일들은 당신의 타래소식(RSS피드)에도 실리게 되므로, 어떤 형식의 파일도 쉽게 포드캐스팅할 수 있게 됩니다.
